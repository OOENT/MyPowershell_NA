MyPowershell_NA

Utilitaire de gestion avancée de scripts PowerShell avec création de journaux de log.
Cet utilitaire permet de centraliser vos scripts PowerShell dans un dossier spécifique, conçu pour être partagé et accessible via le réseau.
Ainsi, plusieurs utilisateurs peuvent exploiter l'outil simultanément.
Pour démarrer l’utilitaire, exécutez le fichier _Lanceur.bat.
Les scripts que vous concevez seront automatiquement stockés dans le dossier .\script
L'utilitaire offre la capacité d'enregistrer les opérations effectuées par chaque script lors de son exécution à travers l’interface de l’utilitaire.
Il archive le code exécuté ainsi que les résultats obtenus, jusqu'à la fermeture de la fenêtre PowerShell.
Il est également possible d'apporter des modifications aux scripts existants.
Lors de chaque modification, une copie de sauvegarde est créée dans le dossier .\log\ScriptHistory, permettant ainsi un suivi précis des changements.

Pour une prise en main intuitive, nous vous invitons à interagir avec l'interface, à lancer des scripts et à consulter les journaux d'activités. Cela vous donnera une vision claire et précise des fonctionnalités de l’outil et de la nature des informations enregistrées.
